Introduction to Python Programming

Course Content

  • Concept of Variables, Data Types
  • Data Structures [List, Dictionary, Set]
  • Conditional Statements
  • Loop
  • Function
  • Introduction to Pandas
  • Exploratory Data Analysis in Pandas
  • Data Visualization
  • Basic Natural Language Processing using Spacy
  • Capstone Project and Review

Logistics

It will be a 2.5 months long course. 1.5 hrs per week. Total 15 hours. It will be offered online via Zoom. Sessions will be recorded.
